Official website for **Ngomane Investment Group**, a South African advisory firm specializing in **Enterprise AI Strategy, Applied Automation, and Governance (GRC)**.

This project delivers a modern, responsive consulting-grade web experience designed to communicate enterprise services while maintaining high performance and minimal technical complexity.

---

# Overview

Ngomane Investment Group bridges the gap between **advanced AI innovation and regulatory compliance** for organizations operating in South Africa.

The website highlights the firm's three core divisions:

• AI Readiness Assessment
• Applied Automation Deployment
• AI Governance & Risk Control

The design philosophy follows modern consulting websites used by top advisory firms and enterprise technology companies.

---

### Consulting-Grade UX

# Architecture Overview

The website follows a **modular section structure**:

Hero Section
↓
Core Divisions
↓
Pricing / Investment Tiers
↓
Contact / Call-To-Action
↓
Footer

Each section is built using reusable components such as:

* service-card
* price-card
* section
* container

This approach keeps the layout scalable and easy to expand.

---

# UI Interaction Logic

JavaScript powers several interactive elements:

### Mouse Spotlight Effect

Service cards display a radial highlight that follows the cursor.

### Mobile Navigation

A responsive hamburger menu enables smooth mobile browsing.

### Scroll Reveal Animation

Page elements fade in as the user scrolls.

### Smooth Navigation

Anchor links scroll smoothly to sections.

}

* services
* pricing
* company description
* contact information
